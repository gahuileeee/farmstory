관리자 메인

상품현황 + 더보기

상품번호 상품명 구분 가격 재고 등록일
[[${product.prodNo}]] [[${product.prodName}]] [[${product.cateName}]] [[${product.prodPrice}]] [[${product.prodStock}]]

주문현황 + 더보기

주문번호 상품명 판매가격 수량 배송비 합계 주문자 주문일

회원현황 + 더보기

[[${user.uid}]] [[${user.name}]] [[${user.nick}]] [[${user.hp}]] [[${user.email}]] [[${user.grade}]]
